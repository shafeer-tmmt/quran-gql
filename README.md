Quran GraphQL

## Description
Added grapql layer for all rest apis, exposed in the  https://quran.api-docs.io/v3

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
